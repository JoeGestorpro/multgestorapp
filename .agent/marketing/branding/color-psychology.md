# Color Psychology — Psicologia das Cores

## Visão Geral
Fundamentação psicológica para as cores do ecossistema MultGestor. Cada cor é escolhida estrategicamente para evocar a emoção e percepção corretas no público-alvo.

## Paleta Principal (MultGestor)
| Cor | Hex | Psicologia | Uso |
|-----|-----|-----------|-----|
| Preto azulado | #070a0d | Autoridade, sofisticação, tecnologia | Background principal |
| Dourado | #d4a853 | Luxo, premium, confiança, sucesso | CTAs, badges, destaques |
| Branco | #ffffff | Clareza, verdade, minimalismo | Texto primário |
| Cinza claro | rgba(255,255,255,0.7) | Hierarquia, secundário | Texto secundário |

## Psicologia por Cor
| Cor | Emoção | Aplicação |
|-----|--------|-----------|
| **Preto** | Poder, elegância, premium | Background = autoridade |
| **Dourado** | Sucesso, conquista, valor | CTA = ação premium |
| **Branco** | Pureza, clareza, verdade | Texto = informação clara |
| **Verde** (ClimaGestor) | Saúde, crescimento, confiança | Nicho clínico |

## Contraste e Acessibilidade
- Texto branco em fundo preto: ratio 19:1 (AAA)
- Texto dourado em fundo preto: ratio 8.5:1 (AAA)
- Texto cinza em fundo preto: ratio 7:1 (AA+)
- CTA dourado em fundo preto: máximo contraste visual
