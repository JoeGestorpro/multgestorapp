# Instagram Reels — Estrutura de Reels para SaaS e Nichos

## Visão Geral
Engine de criação de Reels para Instagram focados em conversão, autoridade e crescimento orgânico. Cada Reel é projetado para segurar atenção nos primeiros 3s e gerar ação.

## Estrutura de Reel de Alto Desempenho
```
Hook (0-3s): prende atenção
Corpo (3-20s): entrega valor
CTA (20-30s): ação desejada
```

## Tipos de Reel por Objetivo
| Tipo | Objetivo | Exemplo |
|------|----------|---------|
| Educativo | Autoridade | "3 erros que matam sua agenda" |
| Case | Prova social | "Como essa barbearia faturou R$ 50k" |
| Produto | Conversão | "Veja como funciona o agendamento online" |
| Engajamento | Alcance | "Sua barbearia ainda usa agenda de papel?" |
| Bastidores | Confiança | "Como é nossa equipe por trás do sistema" |

## Hooks que Funcionam
- Números: "3 erros que..."
- Pergunta: "Sua barbearia ainda...?"
- Controvérsia: "Vou falar algo que ninguém fala"
- Curiosidade: "O segredo das barbearias que lotam"
- Identificação: "Se você é barbeiro, isso é para você"

## Formatos de Reel
- **Talking head**: apresentador falando diretamente
- **Tela verde**: demonstrando o produto ao fundo
- **Storytelling**: narrativa com cenas reais
- **Comparação**: antes/depois de usar o sistema

## Legendas
- Primeira linha = hook (captura atenção)
- Texto curto, direto, com emojis
- CTA no final da legenda
- Hashtags: 5-7 relevantes ao nicho

## Frequência Ideal
- Mínimo: 3 Reels por semana
- Ideal: 1 Reel por dia
- Máximo: 2 Reels por dia (não saturar)

## Integração
| Componente | Como se integra |
|------------|-----------------|
| **Viral Hooks** | Hooks usados nos Reels |
| **Content Pillars** | Reels seguem os pilares |
| **Storytelling** | Narrativa nos Reels |
| **Engagement Engine** | Reels geram engajamento |
