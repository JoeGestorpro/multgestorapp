# Viral Hooks — Hooks Virais para Redes Sociais

## Visão Geral
Catálogo de hooks comprovados para gerar viralidade orgânica no ecossistema MultGestor. O hook é responsável por 80% do sucesso de um conteúdo — sem hook, não há retenção.

## Fórmulas de Hook Viral
| Fórmula | Exemplo |
|---------|---------|
| **Número + Promessa** | "3 erros que estão matando sua agenda (e como evitar)" |
| **Pergunta + Dor** | "Sua barbearia ainda perde cliente por telefone?" |
| **Afirmação controversa** | "Agenda de papel deveria ser proibida" |
| **Curiosidade + Nicho** | "O que barbearias de R$ 50k/mês fazem diferente" |
| **Resultado + Tempo** | "Como uma barbearia faturou R$ 50k em 30 dias" |
| **"Pare de..."** | "Pare de perder cliente com agenda desorganizada" |

## Gatilhos nos Hooks
| Gatilho | Aplicação |
|---------|-----------|
| **Identificação** | "Se você é barbeiro..." |
| **Urgência** | "Enquanto você lê isso..." |
| **Exclusividade** | "Pouca gente sabe disso..." |
| **Autoridade** | "Depois de analisar 100 barbearias..." |

## Hooks para Cada Nicho
| Nicho | Hook |
|-------|------|
| BarberGestor | "O maior erro de barbeiros que querem crescer" |
| ClimaGestor | "Procedimento estético com 40% mais reservas" |

## Teste de Hook
Antes de produzir o conteúdo completo:
1. Escrever 10 hooks para o mesmo tema
2. Testar com 3 pessoas do público-alvo
3. Qual gerou mais curiosidade? Esse é o hook
