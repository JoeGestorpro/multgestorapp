# Short Video Structure — Estrutura de Vídeos Curtos

## Visão Geral
Engine de estruturação de vídeos curtos (Reels, TikTok, Shorts) para o ecossistema MultGestor. Vídeos curtos são o formato de maior alcance orgânico em 2025-2026.

## Estrutura Universal (30s)
```
Tempo | Seção | Elementos
0-3s  | Hook | Prender atenção (pergunta, afirmação, visual impactante)
3-10s | Contexto | Apresentar o problema ou situação
10-20s| Solução | Mostrar a resposta (produto, dica, insight)
20-27s| Prova | Resultado, dado, depoimento
27-30s| CTA | Ação desejada (seguir, comentar, link)
```

## Formatos de Vídeo
| Formato | Duração | Ideal para |
|---------|---------|-----------|
| Dica rápida | 15-20s | Educativo, alto engajamento |
| Demonstração | 30-45s | Produto, features |
| Case | 30-60s | Prova social |
| Tutorial | 45-60s | Conteúdo de autoridade |
| Bastidores | 15-30s | Cultura da marca |

## Regras de Produção
- Hook nos primeiros 1-2 segundos (não 3s)
- Texto na tela (muitos veem sem som)
- Ritmo acelerado (corte a cada 3-5s)
- CTA claro no final (visual + falado)
- Legendas automáticas + revisadas

## Mobile-First
- Formato 9:16 vertical
- Texto grande (legível em tela pequena)
- Elementos visuais no centro
- CTA no final ocupa 1/3 da tela

## Checklist Antes de Publicar
- [ ] Hook nos primeiros 2s?
- [ ] Texto na tela?
- [ ] Legendas corretas?
- [ ] CTA claro?
- [ ] Duração < 60s?
- [ ] Formato vertical?
