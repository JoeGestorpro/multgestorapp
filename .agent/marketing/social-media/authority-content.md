# Authority Content — Conteúdo de Autoridade

## Visão Geral
Engine de criação de conteúdo que estabelece o MultGestor como referência em gestão para barbearias e pequenos negócios. Conteúdo de autoridade é o que faz um cliente escolher você em vez da concorrência.

## Tipos de Conteúdo de Autoridade
| Tipo | Formato | Frequência |
|------|---------|-----------|
| Guia completo | Carrossel / Blog | Mensal |
| Dados do produto | Infográfico / Post | Quinzenal |
| Análise de mercado | Artigo / LinkedIn | Semanal |
| Tutorial avançado | Reels / YouTube | Semanal |
| Comparativo | Post / Página | Único |

## Conteúdo de Autoridade para Barbeiros
- "Guia completo para digitalizar sua barbearia em 7 dias"
- "A matemática por trás do precificação de serviços"
- "Como calcular o ticket médio ideal para sua barbearia"
- "O que barbearias de alto faturamento fazem diferente"

## Estrutura de Conteúdo de Autoridade
```
1. Título que promete aprendizado
2. Introdução que contextualiza
3. Conteúdo principal (passo a passo / análise)
4. Dados e fontes (quando aplicável)
5. Aplicação prática no BarberGestor
6. CTA sutil (sem pressão de venda)
```

## Credibilidade
- Usar dados reais do produto (com permissão)
- Citar barbearias reais (com permissão)
- Mostrar números, não opiniões
- Convidar especialistas do nicho para colaborar

## Distribuição
- Blog → LinkedIn → Instagram (adaptado)
- YouTube → Reels → Shorts (um vídeo, 3 formatos)
- Um conteúdo de autoridade por semana
