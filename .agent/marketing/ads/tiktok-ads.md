# TikTok Ads — Anúncios TikTok

## Visão Geral
Engine de anúncios pagos no TikTok para alcançar um público mais jovem e engajado. TikTok é o canal de maior crescimento para descoberta de novos produtos e serviços.

## Formato de Anúncio
| Formato | Duração | Ideal para |
|---------|---------|-----------|
| In-Feed | 15-30s | Conversão direta |
| Spark Ads (post orgânico impulsionado) | 15-60s | Engajamento + autoridade |
| Brand Takeover | 3s | Alto alcance (caro) |
| Hashtag Challenge | — | Viral (caro) |

## Estrutura de Anúncio In-Feed
```
0-3s: Hook forte (pergunta, curiosidade, impacto)
3-15s: Conteúdo (mostrar o produto ou resultado)
15-25s: Prova (depoimento, dado, antes/depois)
25-30s: CTA ("Link na bio" / "Baixe agora")
```

## Público TikTok (BarberGestor)
- Idade: 18-35 anos

- Comportamento: criadores de conteúdo, small business
