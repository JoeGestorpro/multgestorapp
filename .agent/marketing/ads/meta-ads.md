# Meta Ads — Anúncios Facebook e Instagram

## Visão Geral
Engine de criação e otimização de anúncios pagos no ecossistema Meta (Facebook + Instagram) para o MultGestor. Meta Ads é o principal canal de mídia paga para nichos locais.

## Estrutura de Anúncio
```
Criativo (imagem/vídeo 1:1 ou 4:5)
  → Headline (até 40 caracteres)
  → Texto principal (até 125 caracteres)
  → CTA (botão)
  → Link para landing page
```

## Formatos que Funcionam
| Formato | Performance | Indicação |
|---------|-------------|-----------|
| Reels (vídeo 9:16) | ⭐⭐⭐⭐⭐ | Alcance + conversão |
| Feed estático (4:5) | ⭐⭐⭐⭐ | Conversão |
| Stories (9:16) | ⭐⭐⭐ | Engajamento |
| Carrossel | ⭐⭐⭐ | Vários serviços |

## Segmentação (BarberGestor)
| Parâmetro | Configuração |
|-----------|-------------|
| Público | Donos de barbearia, barbeiros |
| Idade | 22-55 anos |
| Gênero | Masculino (predominante) |
| Interesses | Barbearia, cuidados masculinos, empreendedorismo |
| Local | Brasil (cidades específicas) |
| Comportamento | Pequenos empresários, usuários de Instagram |

## Estrutura de Campanha
```
Campanha: Conversão (otimizar por cadastro/trial)
  ├── Conjunto 1: Público frio (interesses)
  │   └── Anúncio 1: Vídeo mostrando o produto
  │   └── Anúncio 2: Depoimento de cliente
  ├── Conjunto 2: Remarketing (visitou site)
  │   └── Anúncio 1: "Ainda pensando?" com oferta
  └── Conjunto 3: Lookalike (clientes atuais)
      └── Anúncio 1: Testemunhal forte
```
