# High Conversion Ads — Anúncios de Alta Conversão

## Visão Geral
Catálogo de padrões e formatos de anúncios comprovadamente eficazes para SaaS, organizados por objetivo e plataforma.

## Padrões de Anúncio que Convertem

### Padrão 1 — Dor + Solução (Universal)
```
Hook: "Perde cliente por causa de agenda?"
Corpo: "BarberGestor resolve com agenda online 24h"
CTA: "Teste grátis por 7 dias"
```

### Padrão 2 — Prova Social (Confiança)
```
Hook: "230 barbearias já usam"
Corpo: Depoimento rápido de cliente real
CTA: "Comece grátis"
```

### Padrão 3 — Demonstração (Produto)
```
Hook: "Veja como funciona em 30s"
Corpo: Screen recording do produto
CTA: "Testar grátis"
```

### Padrão 4 — Oferta Irresistível (Conversão)
```
Hook: "7 dias grátis + setup guiado"
Corpo: "Sem cartão, sem compromisso"
CTA: "Quero testar"
```

### Padrão 5 — Contraste (Antes/Depois)
```
Hook: "Antes vs Depois"
Corpo: Mostrar antes (problema) e depois (solução)
CTA: "Transformar meu negócio"
```

## Estrutura de Anúncio Vencedor
```
Hook (0-2s): prender atenção
Problema (2-5s): nomear a dor
Solução (5-15s): apresentar o produto
Prova (15-20s): resultado ou depoimento
Oferta (20-25s): trial grátis, garantia
CTA (25-30s): ação clara e urgente
```

## Copy para Anúncios
| Objetivo | Texto | Headline | CTA |
|----------|-------|----------|-----|
| Trial | Chega de perder cliente. | Sistema para Barbearia | Teste Grátis |
| Demo | Veja como funciona. | Agenda Online | Ver Demo |
| Lead | Baixe o guia grátis. | Guia de Gestão | Baixar Agora |
| Venda | 7 dias grátis. | Oferta Especial | Começar |

## Teste A/B de Anúncios
Sempre testar:
- Hook (pergunta vs afirmação)
- Formato (vídeo vs imagem)
- CTA (direto vs benefício)
- Público (interesse vs lookalike)
- Oferta (trial vs desconto)
