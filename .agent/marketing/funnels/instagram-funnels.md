# Instagram Funnels — Funis de Conversão no Instagram

## Visão Geral
Engine de funis de conversão especificamente desenhados para o Instagram — da descoberta ao clique no link. O Instagram é o principal canal de aquisição para nichos visuais como barbearias.

## Funis Instagram

### Funil 1 — Reels → Perfil → Link
```
Reels (alcance orgânico)
  → "Link na bio para teste grátis"
  → Perfil com destaque de links
  → Landing page / Trial
```

### Funil 2 — Stories → Link
```
Stories (engajamento)
  → "Arraste para cima" / link sticker
  → Landing page / Guia gratuito
  → Cadastro / Download
```

### Funil 3 — Post → Comentários → DM
```
Post (educativo / autoridade)
  → "Comente 'QUERO' que envio o link"
  → DM automática com link
  → Landing page
```

### Funil 4 — Anúncio → Landing
```
Anúncio Instagram (tráfego pago)
  → Landing page otimizada
  → Trial grátis
```

## Estrutura de Bio
```
[BarberGestor]
Gestão completa para sua barbearia 🚀
Agenda online • Vendas • Relatórios
⬇️ Teste grátis por 7 dias
[link]
```

## Stickers de Conversão
- Link: "Teste grátis" / "Agende demo"
- Enquete: "Qual sua maior dor?" (engaja + dados)
- Pergunta: "Manda sua dúvida" (qualifica lead)
- Contagem regressiva: "Últimos dias do preço de lançamento"

## Métricas
| Métrica | Meta |
|---------|------|
| Reach → Perfil | > 3% |
| Perfil → Link | > 10% |
| Link → Trial | > 5% |
| Stories → Swipe up | > 5% |
| CPM anúncio | < R$ 15 |

## Automação
- DM automática ao comentar palavra-chave
- Resposta automática a perguntas frequentes via DM
- Agendamento de posts e stories

| Componente | Como se integra |

| **WhatsApp Funnels** | DM → WhatsApp |
