# WhatsApp Funnels — Funis de Conversão no WhatsApp

## Visão Geral
Engine de funis de conversão usando WhatsApp como canal principal. WhatsApp é o canal com maior taxa de abertura e conversão para pequenos negócios.

## Funis WhatsApp

### Funil 1 — Pós-Lead (Nutrição)
```
Lead capturado (e-mail + WhatsApp)
  → D+0: "Oi [nome], vi que baixou nosso guia. Posso ajudar?"
  → D+2: Dica de gestão
  → D+5: Convite para trial grátis
  → D+7: "Como está sendo a experiência?"
```

### Funil 2 — Trial
```
Trial iniciado
  → D+0: "Bem-vindo! Precisa de ajuda no setup?"
  → D+1: "Já criou seu primeiro serviço?"
  → D+3: "Viu os relatórios? Mostram seu lucro"
  → D+6: "Seu trial termina amanhã. Quer continuar?"
```

### Funil 3 — Suporte → Venda
```
Cliente pede ajuda no suporte
  → Resolver problema com excelência
  → "Aproveitando, você sabia que o Premium tem [feature]?"
  → Oferecer upgrade com desconto especial
```

## Templates de Mensagem
| Tipo | Template |
|------|----------|
| Boas-vindas | "Oi [nome], bem-vindo ao BarberGestor! 🚀 Em 3 passos você configura tudo. Precisa de ajuda?" |
| Dica | "Dica rápida: sabia que você pode [ação]? Isso ajuda a [benefício]. Quer saber como?" |
| Trial acabando | "Oi [nome], só lembrando que seu teste grátis termina em 2 dias. Quer continuar com a gente?" |
| Conversão | "Parabéns pela assinatura! 🎉 Agora você tem acesso completo a [features premium]" |

## Regras de WhatsApp
- Nunca enviar mais de 1 mensagem sem resposta
- Respeitar horário comercial (8h-20h)
- Oferecer valor antes de pedir algo
- Personalizar sempre (nome, contexto)
- Link de opt-out em toda campanha

## Métricas
| Métrica | Meta |
|---------|------|
| Taxa de abertura | > 90% |
| Taxa de resposta | > 50% |
| Conversão (lead → trial) | > 30% |
| Conversão (trial → pago) | > 25% |
| Opt-out rate | < 2% |
