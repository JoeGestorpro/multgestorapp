# Booking Funnels — Funil de Agendamento Online

## Visão Geral
Engine de funil específico para agendamento online público — o booking é um funil de conversão por si só, levando o cliente final da descoberta ao agendamento confirmado.

## Funil de Agendamento
```
DESCOBERTA
  ├── Google (pesquisa por barbearia)
  ├── Instagram (perfil da barbearia)
  └── Indicação
       ↓
INTERESSE
  ├── Landing page da barbearia
  └── Link direto /agendamento/:slug
       ↓
AGENDAMENTO
  ├── Seleciona serviço
  ├── Escolhe horário
  └── Confirma com dados de contato
       ↓
CONFIRMAÇÃO
  ├── Confirmação por e-mail
  └── Lembrete automático (planejado)
       ↓
PÓS-AGENDAMENTO
  ├── Cancelamento / reagendamento
  ├── Avaliação do serviço
  └── Novo agendamento
```

## Otimização de Cada Etapa
| Etapa | O que otimizar | Como |
|-------|---------------|------|
| Descoberta | SEO local, Google Meu Negócio | Palavras-chave locais |
| Landing | Velocidade, design, CTA | UI premium, carregamento rápido |
| Seleção | UX, poucos cliques | 3 passos: serviço → horário → confirmar |
| Confirmação | Clareza, confiança | E-mail imediato, dados claros |
| Pós | Facilitar retorno | Link para novo agendamento no e-mail |

## Métricas do Booking Funnel
| Métrica | Meta |
|---------|------|
| Visit → Iniciou agendamento | > 30% |
| Iniciou → Confirmou | > 70% |
| Confirmou → Compareceu | > 85% |
| Compareceu → Novo agendamento | > 30% |
| Tempo médio de agendamento | < 2 min |

## Redução de Abandono
- Barra de progresso (passo 1 de 3)
- Salvamento dos dados do cliente
- Lembrete automático reduz faltas
- Confirmação por e-mail reforça compromisso

## Integração
| Componente | Como se integra |
|------------|-----------------|
| **SaaS Funnels** | Booking alimenta o SaaS |
| **WhatsApp Funnels** | Confirmação por WhatsApp |
| **Lead Funnels** | Booking gera leads para a barbearia |
