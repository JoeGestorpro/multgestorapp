# SaaS Funnels — Funil de Conversão SaaS

## Visão Geral
Engine do funil específico de SaaS — da descoberta ao upgrade pago. O funil SaaS é diferente de e-commerce: o "quase lá" é o trial, o "fechou" é a assinatura.

## Funil SaaS Clássico
```
Visitante → Lead → Trial Ativo → Trial Engajado → Cliente Pago → Cliente Premium
```

## Etapas do Funil SaaS
| Etapa | Ação do usuário | Nosso objetivo |
|-------|----------------|----------------|
| Visitante | Chegou na landing | Capturar lead ou iniciar trial |
| Lead | Baixou conteúdo | Converter em trial |
| Trial Ativo | Criou conta | Engajar com o produto |
| Trial Engajado | Usou 3+ vezes | Mostrar valor do Premium |
| Cliente Pago | Assinou | Reter e expandir |
| Cliente Premium | Fez upgrade | Reter e virar advogado |

## Onboarding no Trial (Dia 1-7)
| Dia | Ação | Objetivo |
|-----|------|----------|
| 1 | Boas-vindas + setup guiado | Primeira configuração |
| 2 | Criar primeiro serviço | Primeira ação de valor |
| 3 | Convidar colaborador | Provar feature social |
| 5 | Analisar relatório | Mostrar valor dos dados |
| 7 | Sugerir upgrade | Converter |

## Gatilhos de Conversão no Trial
- **Uso**: após 3 agendamentos → "Quer relatórios avançados?"
- **Tempo**: D+5 → "Seu trial termina em 2 dias"
- **Resultado**: ao ver lucro → "Premium desbloqueia ainda mais"
- **Limite**: ao bater limite free → "Faça upgrade para continuar"

## Métricas do Funil SaaS
| Métrica | Definição | Meta |
|---------|-----------|------|
| Visit → Trial | % que inicia trial | > 5% |
| Activation Rate | % que completa onboarding | > 60% |
| Trial → Paid | % que vira pago | > 15% |
| Churn Rate | % que cancela | < 5% mês |
| LTV | Receita por cliente | > 10x CAC |
| NPS | Satisfação | > 50 |
