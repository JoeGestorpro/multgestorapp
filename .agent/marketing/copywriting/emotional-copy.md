# Emotional Copy — Copy Emocional

## Visão Geral
Engine de copywriting que ativa emoções para gerar conexão e ação. A decisão de compra é emocional — a copy emocional é o veículo dessa emoção.

## Emoções-Alvo
| Emoção | Como ativar | Exemplo |
|--------|-------------|---------|
| **Orgulho** | "Sua barbearia merece profissionalização" | "Sua barbearia merece o melhor" |
| **Alívio** | "Chega de dor de cabeça com agenda" | "Respire: sua agenda está organizada" |
| **Confiança** | "Dados seguros, suporte humano" | "Seus dados protegidos, seu negócio crescendo" |
| **Pertencimento** | "Junte-se às melhores barbearias" | "Faça parte do seleto grupo que já cresce" |
| **Realização** | "Ver seu negócio crescer organizado" | "Imagina abrir o dashboard e ver tudo no verde" |

## Técnicas de Copy Emocional

### Contraste Emocional
```
"Lembra da sensação de perder um cliente por falta de agenda?
Agora imagina ver sua agenda cheia, clientes confirmados,
faturamento crescendo. Essa sensação é possível."
```

### Projeção
```
"Imagine em 30 dias:
✓ Clientes agendando sozinhos online
✓ Barbeiros com horários organizados
✓ Caixa fechado sem erro
✓ Você trabalhando menos e ganhando mais"
```

### Identificação
```
"Se você é barbeiro, sabe o que é:
- Trabalhar 12h por dia
- Não saber quanto vai ganhar no mês
- Perder cliente porque a agenda bagunçou
- Não ter tempo para família"
```

### Storytelling Emocional
```
"João era barbeiro há 10 anos. Trabalhava muito,
mas o dinheiro não sobrava. Até que ele percebeu:
não era falta de cliente, era falta de organização.
Com o BarberGestor, ele organizou a agenda, 
criou um sistema de vendas e hoje fatura 40% mais.
E o melhor: trabalha 8h por dia."
```

## Gatilhos Emocionais na Copy
| Gatilho | Frase |
|---------|-------|
| Identificação | "Você já passou por isso?" |
| Aspiração | "Sua barbearia pode ser top" |
| Medo | "Enquanto você espera, concorrentes avançam" |
| Alívio | "Chega de complexidade" |
| Realização | "Você merece esse resultado" |

## Regra de Ouro
- 80% emoção positiva
- 20% emoção negativa (só para identificar a dor)
- Sempre terminar em emoção positiva
