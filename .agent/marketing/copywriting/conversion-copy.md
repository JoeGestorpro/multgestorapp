# Conversion Copy — Copy de Conversão

## Visão Geral
Engine de copywriting focada exclusivamente em conversão — seja cadastro, compra, agendamento ou qualquer ação desejada. Conversion copy é a ponte entre interesse e ação.

## Princípios de Conversion Copy
- **Uma ação**: cada página tem uma ação principal
- **Sem distração**: links, menus e elementos que desviam do CTA
- **Benefício sobre feature**: "não perca clientes" > "agenda online"
- **Urgência sutil**: "comece agora" > "cadastre-se"

## Estrutura de Página de Conversão
```
1. Headline de benefício
2. Subheadline de suporte
3. 3 bullets de diferenciais
4. Prova social (número)
5. CTA primário (risco zero)
6. Garantia (uma linha abaixo do CTA)
```

## Copy de Cadastro
- Título: "Comece grátis por 7 dias"
- Sub: "Sem cartão de crédito. Cancele quando quiser."
- Campos: Nome, E-mail, Senha (3 campos apenas)
- Botão: "Criar minha conta grátis →"
- Link: "Já tem conta? Fazer login"

## Copy de Checkout (upgrade)
- Título: "Seja Premium"
- Bullets: o que desbloqueia
- Preço destacado
- Garantia: "Cancele quando quiser"
- Botão: "Quero ser Premium"

## Gatilhos na Conversion Copy
| Elemento | Gatilho |
|----------|---------|
| "Grátis por 7 dias" | Risco zero |
| "Sem cartão de crédito" | Zero compromisso |
| "Cancele quando quiser" | Confiança |
| "Começar agora" | Urgência |
| "230 barbearias" | Prova social |

## Micro-Conversões
| Ação | Copy | Onde |
|------|------|------|
| Ver preços | "Ver planos e preços →" | Hero, menu |
| Ver demo | "Ver demonstração" | Hero |
| Saber mais | "Como funciona →" | Features |
| Começar trial | "Testar grátis" | Todo CTA primário |

## Teste A/B de Conversion Copy
Sempre testar:
- "Teste grátis" vs "Comece grátis"
- "Cadastre-se" vs "Criar conta"
- "Assinar" vs "Quero Premium"
- "Comprar" vs "Investir no meu negócio"
