# Offer Copy — Copy de Oferta

## Visão Geral
Engine de copywriting para apresentação da oferta do MultGestor. A copy da oferta conecta a dor do cliente com a solução do produto, criando desejo e urgência.

## Estrutura da Copy de Oferta
```
1. Headline (promessa principal)
2. Subheadline (dor + solução)
3. Bullet points (3-5 benefícios específicos)
4. Prova social (número ou depoimento curto)
5. Preço (valor seguido de benefício)
6. Garantia (risco zero)
7. CTA (ação clara)
```

## Template de Oferta
```
HEADLINE: O Sistema que Barbearias Premium Usam
SUB: Chega de perder cliente com agenda bagunçada
BULLETS:
  • Agenda online 24h — cliente marca sem ligar
  • Vendas e caixa organizados em um clique
  • Relatórios que mostram exatamente quanto você ganha
PROVA: "230 barbearias já profissionalizaram a gestão"
PREÇO: A partir de R$ 0/mês — teste grátis por 7 dias
GARANTIA: Cancele quando quiser, sem multa
CTA: Começar Grátis →
```

## Copy de Oferta para Cada Canal
| Canal | Adaptação |
|-------|-----------|
| Landing page | Oferta completa com todos os elementos |
| Anúncio | Headline + 1 benefício + CTA |
| E-mail | Oferta com storytelling + bullets |
| Instagram | Oferta resumida em 3 bullets |

## Gatilhos na Copy de Oferta
| Elemento | Gatilho |
|----------|---------|
| "230 barbearias" | Prova social |
| "7 dias grátis" | Risco zero |
| "Sem multa" | Confiança |
| "A partir de R$ 0" | Acessibilidade |

## Variações por Segmento
- **Barbeiro individual**: "Organize sua agenda e cresça sozinho"
- **Barbearia com equipe**: "Gestão completa para sua equipe"
- **Rede de barbearias**: "Escalabilidade para múltiplas unidades"
