# High Ticket Copy — Copy para Ofertas Premium

## Visão Geral
Engine de copywriting para planos premium e ofertas de alto valor do MultGestor. High-ticket não é apenas preço alto — é percepção de valor que justifica o investimento.

## Princípios
- **Valor primeiro**: apresentar todo o valor antes do preço
- **Exclusividade**: plano premium não é para todos
- **Resultado**: vender o resultado, não o software
- **Autoridade**: quem compra premium é tratado como premium

## Estrutura de Copy High-Ticket
```
1. Headline aspiracional ("O plano que barbearias de elite usam")
2. Diferenciais exclusivos (o que só o premium tem)
3. Resultado prometido ("dobre sua eficiência")
4. Prova social de clientes premium
5. Preço (apresentar como investimento, não custo)
6. Bônus que aumentam valor percebido
7. Garantia estendida (reduz risco percebido)
8. CTA com autoridade ("Quero ser Premium")
```

## Features Exclusivas para Copy Premium
- "Agenda online com identidade visual personalizada"
- "Relatórios avançados com gráficos executivos"
- "Suporte prioritário — resposta em minutos, não horas"
- "Landing page personalizada para sua barbearia"
- "Múltiplos colaboradores sem limite"

## Justificativa de Preço
```
"R$ 97/mês = R$ 3,20 por dia.
Isso é menos que um café.
Para um sistema que pode aumentar seu faturamento em 40%."
```

## Objeções High-Ticket
| Objeção | Resposta |
|---------|----------|
| "É caro" | Mostrar ROI + parcelamento |
| "Não preciso" | "Clientes premium usam. E seus concorrentes?" |
| "Depois eu vejo" | Urgência + bônus limitado |
| "Já tenho grátis" | "Free tem limitações que custam caro" |

## Bônus que Funcionam em High-Ticket
- Setup personalizado (consultoria 1:1)
- Suporte prioritário (WhatsApp direto)
- Treinamento da equipe
- Relatórios personalizados
- Migração de dados gratuita
