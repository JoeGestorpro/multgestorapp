# SaaS Copywriting — Copy para Produto SaaS

## Visão Geral
Engine de copywriting específica para produto SaaS — dentro do dashboard, onboarding, e-mails transacionais, notificações e interface do MultGestor.

## Princípios de Copy para SaaS
- **Clareza > Criatividade**: o usuário precisa entender rápido
- **Microcopy importa**: botões, labels, tooltips fazem diferença
- **Tom consistente**: mesma personalidade em toda UI
- **Orientada a ação**: dizer o que fazer, não apenas informar

## Template de Copy para UI
| Elemento | Regra | Exemplo (ruim) | Exemplo (bom) |
|----------|-------|----------------|----------------|
| Botão | Verbo + contexto | "Enviar" | "Confirmar agendamento" |
| Mensagem sucesso | O que aconteceu | "Sucesso" | "Agendamento confirmado! ✅" |
| Mensagem erro | O que houve + solução | "Erro" | "Não foi possível salvar. Tente novamente." |
| Placeholder | Exemplo do que preencher | "Nome" | "Nome do cliente" |
| Tooltip | Contexto extra | "Ajuda" | "Comissão percentual sobre cada serviço" |

## Copy de Onboarding
| Etapa | Copy | Tom |
|-------|------|-----|
| Boas-vindas | "Bem-vindo ao BarberGestor 🚀" | Amigável |
| Primeiro acesso | "Vamos configurar sua barbearia em 3 passos" | Diretivo |
| Tutorial | "Agora vamos criar seu primeiro serviço" | Paciente |
| Sucesso | "Sua barbearia está pronta! 🎉" | Celebratório |

## E-mails Transacionais
| Tipo | Assunto | Tom |
|------|---------|-----|
| Boas-vindas | "Bem-vindo ao BarberGestor!" | Amigável |
| Reset de senha | "Redefina sua senha" | Direto |
| Agendamento confirmado | "Agendamento confirmado ✅" | Claro |
| Trial terminando | "Seu teste grátis termina em 3 dias" | Útil |
| Novo cliente | "Um novo cliente marcou horário!" | Animado |

## Microcopy para Conversão no SaaS
- Upgrade: "Desbloqueie relatórios avançados →"
- Feature locked: "Disponível no plano Premium"
- Empty state: "Nenhum agendamento ainda. Crie o primeiro!"
- Confirmação: "Tem certeza que deseja excluir?" (com nome do item)

## Integração
| Componente | Como se integra |
|------------|-----------------|
| **Tone of Voice** | Tom consistente no produto |
| **Conversion Copy** | Microcopy de conversão |
| **Offer Copy** | Copy de upgrade dentro do produto |
