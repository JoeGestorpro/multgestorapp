# Pain Driven Copy — Copy Orientada a Dores

## Visão Geral
Engine de copywriting que usa as dores do cliente como motor da comunicação. Identificar, nomear e agravar a dor — para então apresentar a solução.

## Dores do Público (BarberGestor)
| Dor | Intensidade | Como usar na copy |
|-----|------------|-------------------|
| Perder cliente por falta de agenda | Alta | "Cliente ligou e você não tinha horário — e ele não volta" |
| Agenda de papel bagunçada | Alta | "Rabiscos, borrachexas, nomes ilegíveis" |
| Não sabe quanto ganha no mês | Média | "Você trabalha mas não sabe se está tendo lucro" |
| Barbeiro esquece horário | Alta | "Barbeiro não apareceu e cliente foi embora" |
| Cliente marca e não vai | Alta | "Cliente agendou e não apareceu — prejuízo na certa" |

## Estrutura Pain → Solução
```
DOR: "Você já perdeu cliente porque a agenda estava bagunçada?"
AGRAVA: "Agora imagina perder 5 clientes por semana — R$ 500 de prejuízo"
SOLUÇÃO: "Com o BarberGestor, sua agenda é digital, automática, sem erros"
RESULTADO: "Cliente marca online, você confirma com 1 clique, ninguém perde horário"
```

## Copy de Dor para Cada Etapa
| Etapa | Foco |
|-------|------|
| Headline | Dor principal |
| Subheadline | Agravamento da dor |
| Features | Como cada feature resolve uma dor |
| Depoimento | "Antes eu [dor], agora [solução]" |

## Gatilhos de Dor
- Identificação: "Se você é barbeiro, já passou por isso"
- Agravamento: "E não é só isso — piora quando..."
- Contraste: "Antes vs Depois"
- Urgência: "Enquanto você lê isso, está perdendo cliente"
