# High Conversion Layouts — Layouts de Alta Conversão

## Visão Geral
Catálogo de layouts comprovados para landing pages SaaS, organizados por objetivo e nicho. Cada layout segue padrões de alta conversão testados em milhares de páginas.

## Layout 1 — Clássico Premium
```
[Hero full-screen com background visual]
[Logos de clientes]
[Features em grid 3 colunas]
[How it works — 3 passos]
[Depoimento em destaque]
[Pricing 3 colunas]
[FAQ]
[CTA final]
```
**Melhor para:** Módulo principal (BarberGestor)

## Layout 2 — Vídeo/Demo
```
[Hero com CTA + thumbnail de vídeo]
[Features em lista vertical]
[Comparação antes/depois]
[Depoimento em vídeo]
[Pricing 2 colunas (simplificado)]
[CTA final com garantia]
```
**Melhor para:** Lançamento de novo módulo

## Layout 3 — Long-form VSL
```
[Hero com headline + CTA]
[Problema → Solução → Features]
[Prova social entre seções]
[FAQ durante o scroll]
[Pricing no meio + no final]
[CTA flutuante sempre visível]
```
**Melhor para:** High-ticket / planos premium

## Layout 4 — Mobile-First
```
[Hero vertical com CTA full-width]
[Features em accordion]
[Depoimentos em swipe]
[Pricing em carrossel]
[CTA sticky bottom]
```
**Melhor para:** Tráfego mobile > 70%

## Princípios Universais
- **Z-pattern**: olho escaneia em Z, layout segue esse padrão
- **F-pattern**: texto escaneado em F, headlines em F
- **White space**: 40% de espaço vazio = premium
- **Contraste**: CTA sempre com maior contraste da página
- **Direção**: setas, olhares, elementos apontam para o CTA
