# Metadata Engine — Engine de Metadados

## Visão Geral
Engine de geração de metadados SEO para todas as páginas do ecossistema MultGestor. Metadados consistentes e otimizados são a base do SEO on-page.

## Template de Title Tag
```
Página principal:
  [Palavra-chave principal] | [Marca]
  Ex: "Sistema para Barbearia | BarberGestor"

Página interna:
  [Nome da página] — [Marca]
  Ex: "Planos e Preços — BarberGestor"

Página local:
  [Palavra-chave] em [Cidade] | [Marca]
  Ex: "Sistema para Barbearia em São Paulo | BarberGestor"
```

## Template de Meta Description
```
[Verbo] + [benefício] + [público] + [diferencial] + [CTA]
Ex: "O sistema de gestão completo para sua barbearia. 
Agenda online, vendas, relatórios. Comece grátis por 7 dias."
```

## Open Graph (Facebook, LinkedIn)
```
og:title = Title tag (mesmo)
og:description = Meta description (mesma)
og:image = 1200x630px com identidade visual
og:url = URL canônica
og:type = website
```

## Twitter Card
```
twitter:card = summary_large_image
twitter:title = Title tag
twitter:description = Meta description
twitter:image = OG image (mesma)
```

## Metadados por Tipo de Página
| Tipo | Title | Description | Image |
|------|-------|-------------|-------|
| Landing principal | Sistema para Barbearia | Gestão completa... | Logo + mockup |
| Landing local | Sistema em [Cidade] | Atendendo [cidade]... | Logo + cidade |
| Blog post | Título do post | Resumo do post... | Thumbnail do post |
| Pricing | Planos — [Marca] | Compare os planos... | Tabela de planos |
| FAQ | Perguntas Frequentes | Tire suas dúvidas... | Logo |

## SEO Audit Checklist
- [ ] Title < 60 chars?
- [ ] Meta description < 160 chars?
- [ ] OG tags presentes?
- [ ] Twitter tags presentes?
- [ ] Canonical tag presente?
- [ ] Imagem OG 1200x630?
