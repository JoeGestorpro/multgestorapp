# SEO Copywriting — Copy para SEO

## Visão Geral
Engine de copywriting otimizada para mecanismos de busca, sem sacrificar a experiência do leitor. SEO copywriting é escrever para humanos de um jeito que Google entende.

## Princípios
- **Natural**: keywords inseridas naturalmente, sem stuffing
- **Valor**: conteúdo útil primeiro, SEO depois
- **Estrutura**: headings, listas, parágrafos curtos
- **Intenção**: responder à pergunta do usuário

## Onde Colocar Keywords
| Local | Prioridade | Regra |
|-------|-----------|-------|
| Title tag | Crítica | Keyword no início |
| H1 | Crítica | Keyword principal |
| Primeiro parágrafo | Alta | Keyword naturalmente |
| H2 | Alta | Keywords secundárias |
| Alt text de imagens | Média | Keyword descritiva |
| URL | Média | Keyword limpa |
| Meta description | Alta | Keyword + CTA |

## Estrutura de Conteúdo SEO
```
H1: Palavra-chave principal
  Introdução (keyword + contexto)
  H2: Benefício 1 (keyword secundária)
    Texto com variações da keyword
  H2: Benefício 2
    Texto com LSI keywords
  H2: FAQ (perguntas reais)
    H3: Pergunta 1?
    H3: Pergunta 2?
  H2: Conclusão com CTA
```
