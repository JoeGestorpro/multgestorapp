# SEO Landing Pages — SEO para Landing Pages SaaS

## Visão Geral
Engine de otimização SEO para landing pages do MultGestor. Cada landing page deve ser encontrada no Google para palavras-chave estratégicas do nicho.

## Palavras-chave Estratégicas
| Nicho | Palavra-chave | Intenção | Volume |
|-------|--------------|----------|--------|
| BarberGestor | "sistema para barbearia" | Compra | Alto |
| BarberGestor | "agenda online para barbearia" | Compra | Médio |
| BarberGestor | "software para barbeiro" | Compra | Médio |
| BarberGestor | "gestão de barbearia" | Pesquisa | Alto |
| ClimaGestor | "sistema para clínica de estética" | Compra | Alto |

## Estrutura SEO da Landing Page
```
Title tag: Palavra-chave principal + Marca | < 60 chars
Meta description: Benefício + CTA | < 160 chars
H1: Palavra-chave principal (1 por página)
H2: Palavras-chave secundárias
H3: Long tail keywords
URL: /sistema-para-barbearia (limpa, com keyword)
```

## On-Page SEO
| Elemento | Regra |
|----------|-------|
| Title | Keyword no início, marca no final |
| Meta description | Benefício claro, CTA, incluir keyword |
| Headings | Hierarquia H1 → H2 → H3, keywords naturais |
| Imagens | Alt text descritivo com keyword |
| Links internos | Para blog, cases, outras páginas |
| Schema | LocalBusiness, Product, FAQ |

## Conteúdo da Landing Page para SEO
- Mínimo 800 palavras na página
- Seções de FAQ com perguntas reais
- Depoimentos com palavras-chave naturais
- Dados e números (Google Featured Snippets)

## SEO Local (BarberGestor)
- Google Business Profile integrado
- Palavras-chave: "sistema para barbearia em [cidade]"
- Depoimentos de barbearias reais (mencionar cidade)
- Schema LocalBusiness com endereço
