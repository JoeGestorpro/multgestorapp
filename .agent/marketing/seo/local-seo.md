# Local SEO — SEO Local para Nichos

## Visão Geral
Engine de SEO local para capturar tráfego de busca geográfica. Barbearias e clínicas são negócios locais — SEO local é o canal de aquisição mais relevante para o MultGestor.

## Estratégia Local
- Landing pages por cidade/região
- Google Business Profile para cada módulo
- Palavras-chave: "sistema para barbearia em São Paulo"
- Backlinks de sites locais (associações, eventos)

## Landing Pages Locais (exemplo)
```
URL: barbergestor.com.br/sistema-para-barbearia-sp
Title: Sistema para Barbearia em São Paulo | BarberGestor
H1: O melhor sistema para barbearia em São Paulo
Conteúdo: menção a bairros, região, dados locais
Schema: LocalBusiness com endereço SP
```
