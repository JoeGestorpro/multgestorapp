# Conversion Psychology — Psicologia da Conversão

## Visão Geral
Engine de princípios psicológicos aplicados à conversão em landing pages, anúncios, e-mails e produto. Cada decisão de compra é emocional — a psicologia da conversão é a ponte entre emoção e ação.

## Princípios de Cialdini Aplicados
| Princípio | Aplicação no MultGestor |
|-----------|------------------------|
| **Reciprocidade** | Oferecer guia gratuito sobre gestão de barbearia |
| **Escassez** | "Vagas limitadas para plano Premium" |
| **Autoridade** | Cases reais, números, selos |
| **Consistência** | Pequeno compromisso (teste grátis) → assinatura |
| **Aprovação social** | "X barbearias já usam" |
| **Afeto** | "Seu negócio merece profissionalização" |

## Gatilhos de Conversão
| Gatilho | Descrição | Onde usar |
|---------|-----------|-----------|
| **Dor** | "Cansado de perder agendamentos?" | Headline, anúncio |
| **Ganho** | "Dobre sua eficiência" | Hero, CTA |
| **Prova** | "Resultado comprovado" | Pricing |
| **Urgência** | "7 dias grátis" | CTA |
| **Exclusividade** | "Seja um dos primeiros" | Lançamento |
| **Curiosidade** | "O segredo das barbearias que lotam" | Anúncio |

## Modelo AIDA para Landing Pages
```
Atenção: Headline poderosa
Interesse: Subheadline com dor ou benefício
Desejo: Features + provas sociais
Ação: CTA claro sem fricção
```

## Objeções Comuns e Como Responder
| Objeção | Resposta | Onde |
|---------|----------|------|
| "É caro" | Mostrar ROI + economia anual | Pricing |
| "Não sei usar" | Setup guiado + suporte | How it Works |
| "Já tenho sistema" | Diferenciais + trial grátis | Features |
| "Não preciso" | Dor ignorada → consequências | Pain Points |
