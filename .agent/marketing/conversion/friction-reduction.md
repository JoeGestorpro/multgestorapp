# Friction Reduction — Redução de Fricção

## Visão Geral
Engine de identificação e eliminação de atritos no funil de conversão. Cada passo extra, cada campo de formulário, cada carregamento lento é fricção que custa conversão.

## Fontes de Fricção
| Fonte | Impacto | Solução |
|-------|---------|---------|
| Formulário longo | Muito alto | Máx 3 campos no cadastro inicial |
| Carregamento lento | Alto | Otimizar imagens, código splitting |
| Cadastro obrigatório | Alto | Google Login + e-mail |
| Preço escondido | Alto | Preço visível antes do CTA |
| Muitas opções | Médio | Uma call to action por vez |
| Texto genérico | Médio | Copy específica por nicho |

## Checklist Anti-Fricção
- [ ] Formulário de cadastro tem ≤ 3 campos?
- [ ] Preço está visível antes de qualquer formulário?
- [ ] CTA primário está acima da dobra?
- [ ] Landing carrega em < 2s?
- [ ] É possível testar sem cartão de crédito?
- [ ] O texto explica o benefício em 5s?

## Mobile Friction
- Formulários com campo único por etapa
- Autocomplete ativado
- Botões com 48px+ de altura
- Sem hover states que não funcionam
- CTA sticky sempre acessível

## Redução de Fricção por Etapa
| Etapa | Fricção | Solução |
|-------|---------|---------|
| Landpage | Muita informação | Uma mensagem por vez |
| Cadastro | Muitos campos | Google + e-mail |
| Pricing | Preço assusta | Trial grátis + parcelamento |
| Checkout | Cartão obrigatório | PIX + boleto + cartão |

## Integração
| Componente | Como se integra |
|------------|-----------------|
| **CTA Engine** | CTA sem fricção |
| **Mobile Conversion** | Fricção mobile reduzida |
| **Conversion Psychology** | Risco zero = fricção zero |
| **Offer Structure** | Oferta clara = menos fricção |
