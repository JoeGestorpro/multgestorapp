# High Conversion Patterns — Padrões de Alta Conversão

## Visão Geral
Catálogo de padrões comprovados de alta conversão para SaaS, organizados por objetivo e contexto. Cada padrão é um template testado que pode ser adaptado ao MultGestor.

## Pattern 1 — Risco Zero Reverso
```
Em vez de: "Teste grátis por 7 dias"
Use: "Comece agora. Se em 7 dias não ver resultados, cancele."
Resultado: maior percepção de confiança
```

## Pattern 2 — Oferta Irresistível
```
1. Bônus: Guia gratuito + template de precificação
2. Garantia: 7 dias de teste sem compromisso
3. Preço: parcelamento sem juros
4. Urgência: bônus por tempo limitado
```

## Pattern 3 — Prova Antes do Preço
```
Sequência: Hero → Features → Depoimento → Pricing
Motivo: visitante precisa confiar antes de ver preço
```

## Pattern 4 — CTA Duplo
```
Hero: CTA principal (teste grátis)
Final da página: CTA de reforço (mesma ação)
Entre eles: CTA "Ver planos" (menos compromisso)
```

## Pattern 5 — Objeção no FAQ
```
P: "Precisa de cartão de crédito?"
R: "Não. Você testa 7 dias grátis sem cadastrar cartão."
Colocar FAQ ANTES do pricing, não depois.
```

## Pattern 6 — Before/After
```
Antes: agenda de papel, cliente ligando, perda de horário
Depois: agenda digital, cliente agendando online, 0 perda
Visual: split screen ou slider de comparação
```

## Pattern 7 — Social Wave
```
"X pessoas estão testando o BarberGestor AGORA"
Contador em tempo real (se tiver dados) ou:
"X barbearias já se cadastraram esse mês"
```

## Testes Recomendados
| Teste | Padrão A | Padrão B | Meta |
|-------|----------|----------|------|
| CTA Hero | "Teste grátis" | "Quero testar" | +10% CTR |
| Pricing | 3 planos | 2 planos + personalizado | +15% conversão |
| Prova | Depoimento texto | Depoimento vídeo | +20% confiança |
