# Como executar a missão "Governança Documental do Obsidian"

1. Localize o prompt completo em: `prompts/executor/governanca-obsidian.md`
2. Copie TODO o conteúdo do prompt
3. Leve para a sala do Executor
4. Execute em modo build
5. Após executar, volte ao chatJoe com o relatório para:
   - chatJoe analisar relatório executor
   - Registrar decisões
   - Compactar
