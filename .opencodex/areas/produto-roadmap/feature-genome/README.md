## O que é

O **Feature Genome** é o **DNA de cada funcionalidade**. Enquanto o Digital Twin mostra a visão macro de um módulo inteiro, o Feature Genome mergulha em cada funcionalidade individual, mapeando:

```
Feature → Produto → Fluxo → Banco → API → Frontend → Backend
  → UX → Analytics → Deploy → Roadmap → PRD → Testes
  → Riscos → Auditorias → Responsáveis → Agentes → Skills
```
