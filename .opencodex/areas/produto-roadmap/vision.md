## Visão Geral

MultGestor é um **core compartilhado multi-tenant** que orquestra **módulos verticais por nicho**. Não é um app — é um **Sistema Operacional de Negócio**.

## Visão de Longo Prazo

> Ser o sistema operacional de gestão para pequenos negócios no Brasil e América Latina.

## Como chegamos lá

1. **BarberGestor** — vertical de prova (atual, em produção)
2. **Fundação Segura** — tornar o Core robusto, seguro e vendável
3. **Segundo vertical** — BeautyGestor/ClimaGestor como prova de reuso
4. **Expansão nichos** — PetGestor, AutoGestor, outros
5. **Expansão geográfica** — Portugal, LATAM

## Princípios de Produto

- Core compartilhado: lógica comum nunca duplicada por nicho
- API-first: tudo via REST
- Event-driven: ações relevantes emitem eventos imutáveis
- Mobile-first progressive: começa web, expande para mobile
- Vertical sobre core: cada nicho é um tema sobre o mesmo motor
