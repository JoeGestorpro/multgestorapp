# Objetivo do Projeto — Ideias

## Objetivo principal

Manter um registro organizado de ideias de melhoria para não perder pensamentos bons e poder priorizar depois.

## Metas atuais

- [ ] capturar primeira ideia
- [ ] classificar ideias por projeto e prioridade
- [ ] revisar periodicamente para levar para execução

## Critérios de sucesso

- ideias não ficam perdidas em conversas
- dá pra revisar e priorizar facilmente
- ideias boas viram missões reais no chatJoe

## Fora do escopo

- executar as ideias (isso é missão separada)
- detalhamento técnico profundo

## Próximo marco

Registrar a primeira ideia de melhoria.
