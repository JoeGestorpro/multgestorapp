# Objetivo do Projeto — Instrutor Gerador de Nichos

## Objetivo principal

Criar um sistema de planejamento de nichos que permita a Joe pensar como arquiteto sênior antes de enviar qualquer implementação para o Code.

## Metas atuais

- [ ] 4 documentos criados na área Nichos/
- [ ] Skill registrada no chatJoe
- [ ] Primeiro nicho planejado usando o instrutor
- [ ] Pipeline humano → IA → humano funcionando

## Critérios de sucesso

- Joe não envia mais prompts genéricos para a IA
- Nichos novos não duplicam recursos do core
- Decisões de arquitetura são registradas e rastreáveis
- IA operacional gera plano antes de implementar

## Fora do escopo

- Automação que gera template automaticamente
- Script de validação do contrato de implementação
- App web do instrutor

## Próximo marco

Criação da área Nichos/ no OpenCodex + skill registrada.
