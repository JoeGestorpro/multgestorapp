## Visão

FiscalGestor será o vertical de **gestão fiscal e contábil** para pequenos negócios. Diferente dos outros nichos (baseados em agenda), este foca em:
- Emissão de notas fiscais
- Controle de tributos
- Conciliação fiscal
- Relatórios contábeis

## Arquitetura Potencial

- Core: Multi-tenant, Auth, Billing, Storage
- Específico: Módulo fiscal, integração com SEFAZ, cálculo tributário
