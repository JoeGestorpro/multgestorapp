## Ciclo de vida do Living OS

```
Missão concluída
  → atualizar scorecards
  → atualizar painel executivo
  → atualizar riscos
  → atualizar próxima ação
  → atualizar estado do projeto
```

Cada missão **termina** atualizando o Living OS se houver mudança em qualquer indicador.
